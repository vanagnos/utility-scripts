scripts for generating environment files from 'openstack environment show' command output
using existing environment files as templates for mapping.



PR_mapper.py: Creates a PR.env file for an instantiated vEIR_PR Stack

base_veir_mapper.py: Creates a base_veir.env file for an instantiated vEIR Stack

veir_conf_mapper.py: Creates a veir_conf.env file for an instantiated vEIR Stack



You can find more information on usage in the first line of each script.
